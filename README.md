# MeguCord-Discord-Theme

## 前提条件
このテーマを使うには **BetterDiscord** または **VenCord** が必要です。

## インストール方法
- Discordの設定を開く 
- VencordかBetterDiscordのThemesを選択 
- Open Themes Folderをクリック
- ダウンロードしたMeguCord.theme.cssを開いたフォルダに移動させる 
- Themesでチェックボックスをクリックして有効化させる
